🚀 Creator Proof
Secure Your Digital Creations on the Blockchain

A Flutter-powered mobile app that enables creators to verify, protect, and prove ownership of digital content using cryptographic hashing and blockchain technology.

🌟 Overview

Creator Proof solves a real problem:
👉 Content theft & lack of ownership proof in digital media

With this app, creators can:

Generate tamper-proof proofs
Store them on blockchain
Instantly verify authenticity
✨ Key Features
🔐 Google Authentication – Secure and seamless login
📁 File Upload & Hashing – SHA-256 based unique fingerprint
⛓️ Blockchain Storage – Immutable proof on Polygon
👛 Wallet Integration – WalletConnect support
✅ Proof Verification – Validate authenticity instantly
🌍 Public Explorer – Discover verified content
👤 User Profiles – Manage your digital assets

🧠 How It Works
graph TD
A[Upload File] --> B[Generate SHA-256 Hash]
B --> C[Store Hash on Blockchain]

Layer Technology
Frontend - Flutter

Hashing - SHA-256

⚙️ Getting Started
🔧 Prerequisites
Flutter SDK
Android Studio / Emulator
Firebase Project

git clone https://github.com/Apoorv0770/Creator_Proof.git
cd Creator_Proof
flutter pub get

🔥 Run App
flutter run

🔐 Firebase Setup
Go to Firebase Console
Create project
Add Android app
Download google-services.json
Place inside:
android/app/

📦 Build for Production
flutter build apk --release

⚠️ Common Issues
Flutter not found -> Add Flutter to PATH
Firebase error -> Check config file
Wallet not connecting -> Add valid WalletConnect ID
